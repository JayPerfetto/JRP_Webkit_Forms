Jason Perfetto's Custom Webkit - Forms Demo

A custom "business" application demonstrating multiple features such as User Management, Complex Forms, Data Management & Security, and Reporting.

Instructions for Use:

Please read the information below before using the application.

The navigation bar on the left will grant you access to the various modules within the application. All users can submit a sample form, view a list of all past submissions and their result reports, or edit their own past submissions. Additionally, Administrators can manage application users, and edit any form submission.


This webkit can support an unlimited number of different forms for your users, and allows the definition of highly granular data-access priviliges for your users across the entire application. There is also full support for all other modules or component from the JRP Angular Webkit, such as informational pages. A generic information page has been included in this application as an example.
